Category Tools for WooCommerce simplifies bulk category management, allowing you to add and remove categories from multiple products at once.

== Description ==

Category Tools for WooCommerce is a straightforward plugin designed to enhance the bulk management of product categories in WooCommerce. This tool allows you to quickly add or remove categories from multiple products directly from the WooCommerce product list page.

*Features:*

* Bulk Category Add: Quickly add a category to multiple products at once.
* Bulk Category Remove: Easily remove a category from multiple products.

This plugin is perfect for store managers looking for a simple way to manage product categories without the need for complex tools or configurations.

Category Tools for WooCommerce is an independent plugin and is not affiliated with or endorsed by WooCommerce or Automattic.

*Requires WooCommerce to be installed and active.*

== Installation ==

1. Install the plugin through the WordPress admin interface or upload the `category-tools-for-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the WooCommerce product list page.
4. Select the products you want to modify and choose the desired bulk action (Add to Category or Remove from Category) from the bulk actions dropdown.
